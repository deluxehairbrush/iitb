# iitb
Proof of concept for the bodhicoder internship at IIT Bombay.
Problem statement: Students may wish to practice extra questions or mock tests in their free time. But, bodhibot currently doesn't generate mock tests or practice questions for the students to practice, nor does it refer to any resources for the student to do the same. 

Solution: Update bodhibot so it can generate questions, mock tests, and act as an extra practice ground for the student. It will only provide questions based on the concepts taught in the course till the date the student asks for it, and generate questions in easy, medium or difficult levels. The bodhibot should also provide answers and explain why.
